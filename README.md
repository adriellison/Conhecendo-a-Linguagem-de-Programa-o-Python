# Conhecendo-a-Linguagem-de-Programa-o-Python

## Tipos de dados

Os tipos built-in são:

| Tipo | Palavra reservada |
| - | - |
| Texto | str |
| Númerico | int, float, complex |
| Sequência | list, tuple, range |
| Mapa | dict |
| Coleção | set, frozenset |
| Booleano | bool |
| Binário | bytes, bytearray, momoryview |
